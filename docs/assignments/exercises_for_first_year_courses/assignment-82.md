# ASCII Art of a HEART

Write Python code that prompts the user for a 4-character word and
    creates the following heart in a heart.txt file.

            HEART        HEART
         HEART HEART  HEART HEART
        HEART     HEART     HEART
        HEART              HEART
         HEART            HEART
          HEART          HEART
            HEART      HEART
              HEART  HEART
                 HEART
